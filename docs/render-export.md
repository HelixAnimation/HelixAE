# Render & Export

## Rendering a Shot

When your comp is ready, use HelixAE to add it to the After Effects Render Queue.

---

## Opening the Render Dialog

Click **Render...** in the CEP panel.

---

## The Render Dialog

| Field | Description |
|---|---|
| **Template** | Select an output module template from the dropdown |
| **Add to Render Queue** | Saves the project and adds the current comp to the AE Render Queue |
| **Close** | Close the dialog without rendering |

---

## Export Dialog

For Prism-managed export (saving as a versioned output), the Export dialog adds more options:

| Field | Description |
|---|---|
| **Task** | The Prism task this export belongs to |
| **Use next version** | Automatically increments to the next version number (on by default) |
| **Comment** | Optional comment saved with this export version |
| **Output Module** | Select which render template to use |
| **Local output** | Toggle whether output goes to local or network path |

Click **Add to Render Queue** to save and queue the render.

---

## After Rendering

Once After Effects finishes rendering, the output is saved as a new version in Prism. Other artists will see it in the Footage Tracker the next time they refresh.
