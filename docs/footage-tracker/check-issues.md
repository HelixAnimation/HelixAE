# Check Issues

## What Are Issues?

The Footage Tracker automatically checks your footage for common problems. When issues are found, the **⚠ Check Issues (N)** button appears in the statistics bar showing the number of problems.

---

## What Gets Checked

| Issue | Description |
|---|---|
| **Outdated version** | A newer version of this footage exists in Prism |
| **FPS mismatch** | The footage FPS doesn't match the shot FPS in Kitsu |
| **Frame range mismatch** | The footage frame range doesn't match the shot in Kitsu |
| **Resolution mismatch** | The footage resolution doesn't match expectations |

---

## How to Use It

1. Open the Footage Tracker
2. If the **⚠ Check Issues** button appears, click it to scroll to and highlight the problematic footage
3. Right-click the flagged item to see sync/update options

You can also trigger a manual check by clicking the **Check Issues** button in the CEP panel at any time, without opening the full tracker.

---

## Fixing Issues

| Issue | Fix |
|---|---|
| Outdated version | Right-click → Sync → Version, or use Update All Outdated |
| FPS mismatch | Right-click → Sync → FPS |
| Frame range mismatch | Right-click → Sync → Frame Range |
| Resolution mismatch | Right-click → Sync → Resolution |
