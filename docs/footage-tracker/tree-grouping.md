# Tree Grouping Modes

## Shot-First vs Identifier-First

For **3D Renders** and **2D Renders**, you can choose how footage is grouped in the tree:

- **Group by Shot** (default) — top level is the shot name, then identifier/task underneath
- **Group by Identifier** — top level is the identifier/task name, then shots underneath

This is a per-render-type setting, so you can have 3D Renders grouped one way and 2D Renders grouped another.

---

## How to Toggle

Click the **title bar** of the render group (e.g. the "3D Renders" header row in the tree) to toggle between modes.

---

## When to Use Each Mode

| Mode | Best for |
|---|---|
| **Group by Shot** | Working on a single shot — see all its renders together |
| **Group by Identifier** | Checking a specific render pass across multiple shots |
