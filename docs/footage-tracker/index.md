# Footage Tracker

The Footage Tracker is the main tool in HelixAE. It gives you a live overview of all footage in your AE project — what version each item is on, whether newer versions are available, and whether there are any FPS or frame range issues.

Open it from the CEP panel via **Footage Tracker...** or from the Prism menu.
