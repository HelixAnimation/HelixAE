# Keyboard Shortcuts & Tips

## Keyboard Shortcuts

| Shortcut | Action |
|---|---|
| **Ctrl+Space** | Toggle the search/filter bar (works in Footage Tracker and Import dialog) |
| **Ctrl+Shift+D** | Toggle the debug console in the Footage Tracker |
| **Ctrl+A** | Select all items in the tracker tree |
| **Delete** | Remove selected items from the tracker tree |
| **Enter** | Apply the current search filter |
| **Escape** | Close the search bar |

---

## Search Tips

The search bar supports wildcards:

| Pattern | Matches |
|---|---|
| `beauty*` | Anything starting with "beauty" |
| `*_aux` | Anything ending with "_aux" |
| `sh0?0` | sh010, sh020, sh030... |
| `[abc]*` | Anything starting with a, b, or c |

Search filters across Shot, Identifier, AOV, and Version at the same time.

---

## Multi-Select

You can select multiple items in the tracker tree:

- **Ctrl+Click** — add individual items to selection
- **Shift+Click** — select a range
- Right-click a multi-selection to apply actions to all selected items at once (e.g. update all, sync all)

---

## Always on Top

Click the **📌** pin icon in the Footage Tracker title bar to keep the window above After Effects. This is useful when working between the tracker and your comps. The setting is saved between sessions.

---

## Refresh vs Kitsu Refresh

- Use **Refresh → Footage** for a fast reload (uses cached Kitsu data)
- Use **Refresh → Kitsu** only when you know someone has updated a task or frame range in Kitsu and you need the latest data now
