# HelixAE

HelixAE is a Prism pipeline plugin for Adobe After Effects. It connects AE directly to your Prism project, letting you track footage versions, import renders, manage compositions, and send shots to the render queue — all without leaving After Effects.

---

## Opening the Panel

HelixAE lives inside After Effects as a **CEP panel**. To open it:

1. In After Effects, go to **Window** in the top menu
2. Find **HelixAE** in the list
3. Dock it wherever you like — most artists keep it on the side

---

## The CEP Panel

The panel is your main control hub. It contains the following buttons:

| Button | What it does |
|---|---|
| **Save Version** | Save the current AE project as a new Prism version |
| **Save Comment...** | Save a new version with an extended comment/note |
| **Project Browser...** | Open the Prism project browser |
| **Footage Tracker...** | Open the Footage Tracker dialog |
| **Import Media...** | Open the Import dialog to bring in renders |
| **Check Issues** | Quickly check for footage problems (outdated versions, FPS/frame range mismatches) |
| **Prism Settings...** | Open Prism settings |
| **Render...** | Open the Render dialog to queue a shot for rendering |

---

## Typical Artist Workflow

1. Open AE and the HelixAE panel
2. Use **Footage Tracker** to check if your renders are up to date
3. Use **Import Media** to pull in new renders
4. Work on your comp
5. Use **Render** to add your shot to the render queue
6. Use **Save Version** to save your progress
